Лабораторная работа по газодинамике
Расчет модального уравнения вида du/dt + a * du/dx = 0
Решение проводилось с использованием трехточечной схемы Лакса и схемы
правый уголок для вычисления скорости в n-ой точке на следующем временном
слое.

Программа производит расчет и записывает результаты в .txt файл.
Для визуализации нужно перенести данные с .txt файла в программу для
построения графиков (например, Grapher 13)

В папке так же присутсвует скриншот с визуализацией обработанных результатов.
